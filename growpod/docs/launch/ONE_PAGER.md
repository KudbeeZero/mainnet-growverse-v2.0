# GrowPod Empire (GROWv2) — One-Pager

**A scientifically grounded cultivation sim whose outputs are unique, verifiable, player-owned
genetics — settled on Algorand.**

---

## The hook

Most grow games are reskinned idle timers over a fixed strain list. GrowPod Empire is a real
quantitative-genetics simulation with a disciplined, auditable economy — where the rare strains you
breed are provably yours, and provably real, because the game that made them is deterministic and
verifiable.

## What it is

A persistent-economy game built on one loop: **grow → care → harvest → cure → sell / breed /
stabilize → mint → trade.** You run grow pods through a server-authoritative simulation, breed a real
14-trait genome with seeded, replayable inheritance, stabilize promising lines so they breed true,
and — for rare results — mint them as blockchain assets.

The design principle that makes it trustworthy: **the database is the source of truth; the chain
mirrors it.** Money is `Decimal`, ledger-posted, and audited, with every faucet matched by a sink.
Every breeding outcome is seeded and disclosed, so anyone can replay a cross and verify it.

## Why Algorand

- **Instant finality**, blocks in roughly under three seconds — one confirmation, no reorgs.
- **Fees around 0.001 ALGO** — a fraction of a cent, so minting small game assets is viable.
- **Native ASA token standard** — NFTs without bespoke contracts.
- **Mature metadata norms** — ARC-3 (with ARC-19 for updatable assets), legible to standard wallets.

## What's mintable

- **Strain NFTs** — stabilized, rarity-gated cultivars carrying genome, lineage, trait ranges,
  stability, and terpene profile as ARC-3 metadata sourced from server truth.
- **Harvest NFTs** — premium harvests with weight, quality, and cannabinoid readings.
- **Seed / proof-of-play tokens** — anchoring the entropy inputs so genetics are independently
  re-derivable.

*Status today:* the mint path and real Algorand provider are built and run in a default-on **simulate
mode** (logged, not broadcast). Funded TestNet + IPFS metadata and any mainnet settlement are
**planned and gated** — no real-money settlement is live.
