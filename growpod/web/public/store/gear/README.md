# Store gear images

Product art for the **Grow Room Gear** store section (`/store`). Each gear item in
`balance.yaml` (`shop.gear`) has an `image:` filename that the storefront loads from
`/store/gear/<image>`. If a file is missing, the card falls back to a category emoji
(💡 / 🌀 / 🪴) — so the store works with or without art.

Expected filenames (transparent or white-background PNG, ~square, ≥512px):

Lights
- `led_125w.png`
- `led_320w.png`
- `led_480w.png`
- `led_700w.png`
- `led_800w.png`

Fans
- `clip_fan.png`
- `oscillating_fan.png`
- `inline_exhaust_6in.png`

Soil & amendments
- `worm_castings.png`
- `bat_guano.png`
- `coco_coir.png`
- `super_soil.png`
- `perlite_mix.png`

Source: licensed Adobe Stock product photos (background-removed), or any matching
product art. Keep them lightweight.
